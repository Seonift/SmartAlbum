# SmartAlbum-b

## Prerequisites
- Install CocoaPods to OSX
```
$ gem install cocoapods
```

## Installation
- Move to project directory and run below commands
```
$ cd ./SmartAlbum/TeamB/SmartAlbum
$ pod install
```
- Open ```SmartAlbum.xcworkspace``` and build
